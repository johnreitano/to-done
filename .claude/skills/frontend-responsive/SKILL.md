## When to use this skill

- When using Tailwind breakpoint prefixes (`sm:`, `md:`, `lg:`, `xl:`)
- When creating responsive grid layouts (`grid-cols-1 md:grid-cols-2`)
- When implementing mobile navigation vs desktop sidebar
- When using `hidden lg:block` or `lg:hidden` patterns
- When ensuring minimum 44x44px touch targets on mobile
- When adjusting typography with responsive sizes (`text-2xl md:text-3xl`)
- When using `container mx-auto` with max-width constraints
- When testing layouts in browser DevTools device emulation
