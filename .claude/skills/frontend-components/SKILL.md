## When to use this skill

- When creating or editing `components/**/*.tsx` files
- When defining TypeScript interfaces for component props
- When deciding whether to use `'use client'` directive
- When organizing hooks at the top of components
- When implementing composition patterns (`<Card><CardHeader>`)
- When managing component state with `useState`
- When lifting state up to parent components
- When adding optional `className` prop for style customization
- When structuring event handler functions
