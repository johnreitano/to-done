## When to use this skill

- When creating new files or directories in the project
- When organizing code in `app/`, `components/`, `lib/`, `hooks/`, `types/`
- When creating or editing `.env.local` or `.env.example`
- When using `NEXT_PUBLIC_` prefix for client-exposed environment variables
- When adding dependencies with `pnpm add`
- When deciding where to place new utility functions or components
- When setting up `next.config.js`, `tsconfig.json`, or other config files
- When structuring Supabase or Stripe integration code in `lib/`
