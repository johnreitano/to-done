## When to use this skill

- When adding `className` attributes with Tailwind CSS utilities
- When using `cn()` from `@/lib/utils` for conditional classes
- When editing `tailwind.config.ts` for custom theme values
- When organizing classes: layout, spacing, sizing, typography, colors, effects
- When styling Radix UI components with Tailwind
- When implementing `hover:`, `focus:`, `active:` states
- When using design tokens like `bg-primary` or `text-gray-900`
- When avoiding `@apply` and custom CSS files
