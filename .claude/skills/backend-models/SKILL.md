## When to use this skill

- When designing new database table schemas
- When defining relationships between tables (foreign keys)
- When creating or editing `types/database.ts`
- When running `supabase gen types typescript`
- When adding `created_at` and `updated_at` columns
- When choosing between TEXT, VARCHAR, UUID, TIMESTAMPTZ data types
- When planning table indexes for foreign keys
