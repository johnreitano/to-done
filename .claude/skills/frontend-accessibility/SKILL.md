## When to use this skill

- When using Radix UI components (Dialog, DropdownMenu, Popover, Select, Toggle)
- When adding `aria-label`, `aria-describedby`, or other ARIA attributes
- When implementing keyboard navigation for interactive elements
- When managing focus in modals, dialogs, or dropdown menus
- When adding alt text to `<Image>` components
- When ensuring color contrast meets WCAG standards
- When using semantic HTML (`<nav>`, `<main>`, `<button>`, `<form>`)
- When testing with VoiceOver, NVDA, or browser accessibility tools
